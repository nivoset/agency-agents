# Diplomatic Response Crafter References

This directory contains reference materials for the Diplomatic Response Crafter agent, extracted from Dale Carnegie's "How to Win Friends and Influence People."

## Files

| File | Description |
|------|-------------|
| `00-principles-overview.md` | Quick reference table of all 30 principles with application notes |
| `01-fundamental-techniques.md` | Part One: The three foundational principles (Don't criticize, Appreciate, Arouse want) |
| `02-winning-people-to-your-way.md` | Part Three: 12 principles for persuasion without confrontation |
| `03-changing-people-without-offense.md` | Part Four: 9 principles for correction and boundary-setting |
| `04-quick-reference-card.md` | Rapid checklist for crafting responses |

## Usage

The agent should reference these materials when:

1. **Selecting principles**: Use `00-principles-overview.md` to match comment types to appropriate principles
2. **Understanding depth**: Use `01-03` files for the "why" behind each principle with stories and examples
3. **Quick checks**: Use `04-quick-reference-card.md` as a pre-send verification checklist

## Source

All content is derived from Dale Carnegie's "How to Win Friends and Influence People" (1936, revised editions). The principles are:

- **Part One** (3 principles): Fundamental Techniques in Handling People
- **Part Two** (6 principles): Six Ways to Make People Like You
- **Part Three** (12 principles): How to Win People to Your Way of Thinking
- **Part Four** (9 principles): Be a Leader: How to Change People Without Giving Offense

For the Diplomatic Response Crafter, Parts One, Three, and Four are most relevant. Part Two principles (making people like you) are secondary but useful for building rapport.
