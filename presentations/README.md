# Presentations Division

This division contains specialists focused on talk design, slide storytelling, live-delivery-first presentation craft, and presentation critique.

## Agents

- `presentation-story-architect.md` - builds conference-quality talks, slide plans, and speaker notes
- `presentation-critique-director.md` - critiques structure, clarity, proof, pacing, and persuasion
- `presentation-audience-skeptic.md` - pressure-tests likely objections, distrust, boredom, and audience resistance
- `presentation-web-deck-auditor.md` - reviews browser-viewable decks, exported files, and linked web evidence
